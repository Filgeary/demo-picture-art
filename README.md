# demo-picture-art

> JS training-project from Udemy course
> 
> legacy markup as is

## Features

- Modals
- Animations
- Sliders
- Forms
- Input `tel` mask
- Get Cards via `fetch`
- Calculator
- Filter tabs
- Accordion
- Dropdown Menu
- Smooth scrolling & Scroll to top
- Drag&Drop for file upload

## Performance

- `content-visibility: auto` in CSS for fast rendering
- native `lazy loading`
- `webp` & `avif` images for first 2 screens
